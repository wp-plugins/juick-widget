Juick widget

== Description ==

Juick widget displays you latest Juick.com messages in any place in your WordPress theme.

== Installation ==

1. Upload `juick` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Add Juick widget in widgets page.
